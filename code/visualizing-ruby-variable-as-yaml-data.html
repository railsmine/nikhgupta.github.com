Visualizing Ruby variable as YAML data

06 July 2011

Since, I am learning Ruby these days, I needed something like the var_dump function, available in PHP. Now, ruby does have a much nicer feature for this, but I needed something that allowed me to view the tree structure (as possible with using xdebug on PHP). So, I came up with the following function, which allows me to dump a variable using YAML format.

Usage

This function can be used as below:

  • var_dump(somevar) :: dump variable on console, with YAML format
  • var_dump(somevar, "f") :: dump variable in file, with YAML format
  • var_dump(somevar, "c") :: dump variable on console, no formatting done

Code Snippet

def var_dump(var,type="")
  if type=="c"
    ## will dump formatted YAML data to console
    print(var)
  elsif type=="f"
    ## will dump formatted YAML data to a file
    path = File.join(dirname(__FILE__), "var_dump.yml")
		File.open(path, 'w') {|f| f.print(YAML::dump(var)) }
  else
    ## will dump unformatted variable to console
    print (YAML::dump(var))
  end
end
